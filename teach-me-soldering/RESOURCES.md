# Soldering with Flux Resources

## Knowledge

- [Chemtronics: Essential Guide to Flux for Soldering Electronics](https://www.chemtronics.com/essential-guide-to-flux-for-soldering-electronics)
  Comprehensive reference on flux types (rosin, no-clean, water-soluble), chemistry, and selection criteria. Use for: understanding what flux does and which type to use when.

- [SparkFun: How to Solder -- Through-Hole Soldering](https://learn.sparkfun.com/tutorials/how-to-solder-through-hole-soldering/advanced-techniques-and-troubleshooting)
  Step-by-step through-hole technique with photos and troubleshooting. Covers flux application, common defects, and cleanup. Use for: core soldering technique and joint quality diagnosis.

- [PINE64 Wiki: Pinecil Guides to Soldering](https://wiki.pine64.org/wiki/Pinecil_Guides_to_Soldering)
  Official guide covering temperature settings, tip selection, and firmware. Use for: Pinecil V2 setup and temperature calibration.

- [PINE64 Wiki: Pinecil Tips](https://wiki.pine64.org/wiki/Pinecil_Tips)
  Tip types, compatibility, and which tip for which job. Use for: choosing the right tip for ESP32 header work.

- [Oscar Liang: Build a DIY Solder Smoke Extractor](https://oscarliang.com/diy-solder-smoke-extractor/)
  Budget fume extractor build (~$20) using a PC fan and activated carbon filter. Use for: apartment ventilation solution.

- [YouTube: Simple Trick for Soldering Header Pins](https://www.youtube.com/watch?v=SvX63x9rvoU)
  Visual demo of soldering headers straight to Arduino/ESP32 boards using a breadboard as a jig. Use for: header alignment technique.

- [ElectronicsHub: How to Use Solder Flux](https://www.electronicshub.org/how-to-use-solder-flux/)
  Practical step-by-step on flux paste application with photos. Use for: flux application technique.

## Wisdom (Communities)

- [r/soldering](https://reddit.com/r/soldering)
  Active subreddit for sharing joints and getting feedback. Post your joints for critique.

- [r/AskElectronics](https://reddit.com/r/AskElectronics)
  Broader electronics community. Good for "is this joint OK?" and troubleshooting.

- [EEVblog Forum -- Beginners](https://www.eevblog.com/forum/beginners/)
  High-quality electronics forum with experienced members. Good for deeper technical questions.
