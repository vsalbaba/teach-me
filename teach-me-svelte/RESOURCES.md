# Svelte Resources

## Knowledge

- [Svelte Official Interactive Tutorial](https://svelte.dev/tutorial)
  The best hands-on starting point. Teaches Svelte concepts in-browser with live exercises. Use for: reinforcing any concept after an explainer session.

- [Svelte 5 Official Docs](https://svelte.dev/docs/svelte/getting-started)
  Reference documentation for all Svelte APIs including runes ($state, $derived, $props). Use for: looking up specific API behavior.

- [SvelteKit Routing Docs](https://svelte.dev/docs/kit/routing)
  Explains the file-based routing system (+page.svelte, +layout.svelte, etc). Use for: understanding how file structure maps to URLs.

- [SvelteKit Introduction](https://svelte.dev/docs/kit/introduction)
  High-level overview of what SvelteKit adds on top of Svelte. Use for: understanding the difference between Svelte (the component framework) and SvelteKit (the app framework).

- [The Complete Svelte 5 Course -- Joy of Code](https://joyofcode.xyz/learn-svelte)
  Community tutorial covering Svelte 5 with runes. Clear explanations of reactivity and signals. Use for: alternative explanations if official docs don't click.

## Wisdom (Communities)

- [r/sveltejs](https://reddit.com/r/sveltejs)
  Active subreddit with good signal. Use for: troubleshooting, seeing how others structure apps.

- [Svelte Discord](https://svelte.dev/chat)
  Official Discord server. Use for: real-time help, community interaction.
